# ⛑️Help us
-   근처에 있는 사용자 간의 도움을 주고 도움을 받는 앱입니다
-   근처에 있는 사용자 간의 도움을 요청하고 도움을 확인해 보세요
-   채팅을 통해 도움을 해결해 주고 해결 받아보세요
<h2 id="3">앱 기능 설명</h2>

-   로그인/회원가입/로그아웃/읽지 않은 메시지 개수 확인
-   사용자가 도움이 필요한 항목, 상세 내용, 주소, 가격 등을 입력하여 등록할 수 있습니다
-   근처에 있는 다른 사용자만이 등록된 도움 요청 들을 확인할 수 있습니다
-   다른 사용 자가 작성 글을 확인하고 상대방과 채팅을 하여 위치, 가격, 내용에 대해 문의할 수 있습니다
-   나의 작성 글을 확인하고 삭제/완료 할 수 있습니다
