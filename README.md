# ⛑️Help us
-   근처에 있는 사용자 간의 도움을 주고 도움을 받는 앱입니다
-   근처에 있는 사용자 간의 도움을 요청하고 도움을 확인해 보세요
-   채팅을 통해 도움을 해결해 주고 해결 받아보세요
<h2 id="3">앱 기능 설명</h2>

-   로그인/회원가입/로그아웃
-   총 산책 시간, 총 산책 횟수 확인할 수 있습니다
-   산책을 할 때 산책한 시간, 날짜, 거리, 경로를 확인하고 저장할 수 있습니다
-   저장된 정보들을 날짜에 맞춰 확인할 수 있습니다
-   커뮤니티에서 글을 작성할 수 있고 다른 사용자의 글에 댓글을 입력할 수 있습니다
-   나의 정보를 확인하고 수정할 수 있습니다
